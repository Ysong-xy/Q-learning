# 两市商透明市场下的 Q-Learning 收敛结果

## 1. 实验目标

两个实验具有相同的：

```text
市商数量：2
商品价格：[2.0, 2.6]
初始动作：[0, 1]
透明观测结构
```

两组只改变需求分配和折扣因子 `gamma`，观察是否分别收敛到高价和低价。

## 2. 透明公共状态

每个 Agent 都能看到上一轮双方的：

```text
动作
需求
收益
```

因此它们可以区分 `LL`、`HL`、`LH`、`HH`，并根据公共历史学习条件策略。

## 3. 基本收益结构

第一个字母表示自己，第二个字母表示对手：

```text
LL：自己低价，对手低价
HL：自己高价，对手低价
LH：自己低价，对手高价
HH：自己高价，对手高价
```

两个实验都满足：

```text
LH > HH > LL > HL
```

这同时保证：

1. 面对高价对手时，降价具有短期诱惑；
2. 双方高价优于双方低价；
3. 面对低价对手时，低价是最优反应。

## 4. 高价收敛组

```text
gamma = 0.99
LL = 410
HL = 200
LH = 560
HH = 500
```

排序为：

```text
560 > 500 > 410 > 200
```

结果：

```text
final price = 2.594
final profit = 498.49
all-high = 0.986
all-low = 0.007
```

高 `gamma` 使未来低价惩罚足以约束当期降价诱惑，透明历史支持高价协调。

## 5. 低价收敛组

```text
gamma = 0.30
LL = 410
HL = 50
LH = 700
HH = 430
```

排序为：

```text
700 > 430 > 410 > 50
```

结果：

```text
final price = 2.001
final profit = 409.85
all-high = 0.000
all-low = 0.996
```

较低的 `gamma`、很大的当期降价收益以及较小的高价合作增益，使 Agent 收敛到低价竞争。

## 6. 结论

在两组都满足 `LH > HH > LL > HL` 的前提下：

```text
高 gamma + 较温和的降价诱惑
-> 高价收敛

低 gamma + 很强的降价诱惑
-> 低价收敛
```

因此实验差异不是通过破坏基本博弈排序得到的，而是来自未来收益权重和收益差距。
