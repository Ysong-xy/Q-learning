# 两人两价折扣因子对照实验

## 1. 控制变量

两组实验只改变折扣因子 $\gamma$，其余设定完全一致。

| 参数 | 两组共同设定 |
|---|---|
| 卖家数 | $2$ |
| 价格 | $\{2.0,2.6\}$ |
| 成本 | $1.0$ |
| 初始动作 | $(L,H)$ |
| 随机种子 | $2$ |
| 训练轮数 | $50000$ |
| 学习率 | $\alpha_+=0.05,\alpha_-=0.001$ |
| 探索率 | $0.04\to0.002$ |
| 新状态 Q 先验 | 原始实现，先验强度 $1000$ |

收益矩阵在两组中均为

$$R_{LL}=410,\qquad R_{LH}=560,\qquad R_{HL}=200,\qquad R_{HH}=500.$$

唯一不同为

$$\gamma\in\{0.99,0.01\}.$$

## 2. 原始 Q 表初始化

每个状态第一次被访问时，使用原始代码的规则：

| 上一轮状态 | $Q_0(L)$ | $Q_0(H)$ |
|---|---:|---:|
| $LL$ | $0$ | $1000$ |
| $LH$ | $1000$ | $0$ |
| $HL$ | $1000$ | $0$ |
| $HH$ | $0$ | $1000$ |

因此两组实验具有完全相同的初始高价偏置。

## 3. 结果

以最后 $5000$ 轮计算：

| $\gamma$ | 平均价格 | 单个卖家平均利润 | $P(HH)$ | $P(LL)$ | 最终状态 |
|---:|---:|---:|---:|---:|---|
| $0.99$ | $2.594$ | $498.49$ | $0.986$ | $0.007$ | 高价稳定 |
| $0.01$ | $2.001$ | $409.87$ | $0.000$ | $0.996$ | 低价稳定 |

结论：即使使用相同的高价 Q 先验，$\gamma=0.01$ 也不会长期收敛到高价。它在训练前期暂时维持高价，但约在第 $3000$ 轮后开始下降，约在第 $12000$ 轮后基本稳定于低价。

## 4. 原因

Q-learning 目标为

$$y_i=r_i+\gamma\max_{a'}Q_i(s',a').$$

当 $\gamma=0.01$ 时，未来 Q 值几乎不影响当前选择，Agent 主要比较单期收益。对手选高价时，

$$R_{LH}=560>R_{HH}=500,$$

因此随机探索一旦让 Agent 尝试降价，$Q(HH,L)$ 会逐渐高于 $Q(HH,H)$。初始的 $1000$ 偏置只能延缓这一过程，不能改变长期结果。

当 $\gamma=0.99$ 时，Agent 重视降价后的未来低价惩罚，因此高价状态能被维持。
