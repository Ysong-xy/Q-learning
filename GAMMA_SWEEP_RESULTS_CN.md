# 三类市场的 Gamma 对照实验

## 共同设定

- 价格范围统一为 $[2.0,2.6]$；两价组为 $\{2.0,2.6\}$，多价组为 $\{2.0,2.1,\ldots,2.6\}$。
- 每个实验训练 $1000000$ 轮，$\epsilon:0.20\to0.001$。
- $Q_0(s,a)=0$，$\alpha_+=0.05$，$\alpha_-=0.01$。
- 分别测试 $\gamma\in\{0.01,0.30,0.60,0.95,0.99\}$。
- 结果是最后 $50000$ 轮平均值。收敛检查要求相邻两个 $50000$ 轮窗口的平均价格变化 $<0.01$，动作分布 TV 变化 $<0.03$。

## 多人两价（4 人，2 价）

| $\gamma$ | 平均价格 | 平均利润 | 全员同价率 | 收敛 |
|---:|---:|---:|---:|---|
| 0.01 | 2.000 | 249.365 | 0.998 | 是 |
| 0.30 | 2.000 | 249.365 | 0.998 | 是 |
| 0.60 | 2.000 | 249.365 | 0.998 | 是 |
| 0.95 | 2.001 | 249.416 | 0.993 | 是 |
| 0.99 | 2.013 | 250.345 | 0.914 | **否** |

该模型在前四个 $\gamma$ 下收敛到几乎全员低价。$\gamma=0.99$ 仍存在明显的动作分布漂移。

## 两人多价（2 人，7 价）

| $\gamma$ | 平均价格 | 平均利润 | 双方同价率 | 收敛 |
|---:|---:|---:|---:|---|
| 0.01 | 2.001 | 497.745 | 0.996 | 是 |
| 0.30 | 2.001 | 497.765 | 0.995 | 是 |
| 0.60 | 2.002 | 498.341 | 0.988 | 是 |
| 0.95 | 2.277 | 619.134 | 0.075 | **否** |
| 0.99 | 2.349 | 656.162 | 0.313 | **否** |

低和中等 $\gamma$ 收敛到最低价附近。高 $\gamma$ 明显提高了平均价格和利润，但 $0.95$ 与 $0.99$ 在百万轮后仍在多个价格策略间切换，不应宣称已收敛。

## 多人多价（4 人，7 价）

| $\gamma$ | 平均价格 | 平均利润 | 全员同价率 | 收敛 |
|---:|---:|---:|---:|---|
| 0.01 | 2.272 | 299.910 | 0.000 | 是 |
| 0.30 | 2.267 | 299.834 | 0.000 | 是 |
| 0.60 | 2.282 | 304.358 | 0.000 | 是 |
| 0.95 | 2.261 | 297.963 | 0.000 | 是 |
| 0.99 | 2.268 | 299.831 | 0.000 | 是 |

“全员同价率几乎为零”与“收敛”不矛盾：该组稳定在一个非对称的价格动作分布，而不是四人每轮选择完全相同的价格。
